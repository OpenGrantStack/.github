# Contributing to OpenGrantStack

We welcome contributions that improve clarity, security, interoperability, and real-world usability.

## How to Contribute

- Open an issue to discuss changes before large PRs
- Follow repository-specific guidelines
- Write clear commit messages
- Include documentation when changing behavior

## Philosophy

OpenGrantStack values:
- Transparency over cleverness
- Stability over novelty
- Real-world impact over hype

All contributors are expected to follow the Code of Conduct.
